Query will insert the results directly into HDFS:
=================================================
INSERT OVERWRITE DIRECTORY '/path/to/output/dir' SELECT * FROM <table-name> WHERE id > 100;
or
$hive -e "SELECT * FROM table where id > 100" > ~/sample_output.txt

Query will insert the results directly into Local FileSystem:
=============================================================
INSERT OVERWRITE LOCAL DIRECTORY '/path/to/output/dir' SELECT * FROM <table-name> WHERE id > 100;
